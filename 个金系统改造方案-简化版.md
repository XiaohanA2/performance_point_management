# 个金系统改造方案（简化版）

## 一、核心变更

### 1.1 加分业务简化
**变更**：加分业务只做提报通路，不计算积分
- 移除加分业务的计分规则配置（unitPrice、maxScore）
- 移除加分业务的积分计算逻辑
- 保留提报记录和导出功能
- 前端显示提报记录，但不显示分数

**影响范围**：
- 数据库：pf_tasks 表加分业务不再需要 scoreConfig
- 云函数：移除加分业务积分计算
- 前端：工作台加分卡片不显示得分，只显示完成量

### 1.2 必选业务任务分配
**变更**：从按角色配置改为按员工个人配置
- 新增员工表
- 新增员工任务配置表
- 管理后台支持按支行/角色筛选后批量配置
- 支持两种配置模式：
  - 选员工 → 配置多个业务的任务数
  - 选业务 → 配置多个员工的任务数

### 1.3 业务管理
**保留**：必选和加分业务都支持增删改查
- 必选业务：配置权重分、封顶分
- 加分业务：只配置名称、单位、描述

async function getEmployeeTask(userId, taskId, period) {
  const db = uniCloud.database();
  const result = await db.collection('opendb-pf-employee-tasks')
    .where({ employeeId: userId, taskId, period })
    .get();

  return result.data[0]?.target || null;
}

async function calculateRequiredScore(userId, taskId, period) {
  // 1. 获取个人任务数
  const target = await getEmployeeTask(userId, taskId, period);
  if (!target) return 0;

  // 2. 获取完成数
  const totalValue = await getTotalValue(userId, taskId, period);

  // 3. 贡献度计算（组内排名）
  const groupStats = await getGroupStats(userId, taskId, period);

## 六、数据迁移脚本

```javascript
// 迁移脚本：生成员工记录和初始任务配置
async function migrateToEmployeeBasedConfig() {
  const db = uniCloud.database();

  // 1. 从 uni-id-users 生成员工记录
  const users = await db.collection('uni-id-users')
    .where({ role: db.command.in(['manager', 'lobby_manager', 'elastic_counter', 'counter_manager']) })
    .get();

  for (const user of users.data) {
    await db.collection('opendb-pf-employees').add({
      userId: user._id,
      name: user.name,
      branch: user.branchName || user.branch,
      role: user.role,
      status: user.status || 'active',
      createdAt: Date.now()
    });
  }

  // 2. 从 pf_tasks.targetByRole 生成初始任务配置
  const tasks = await db.collection('pf_tasks')
    .where({ category: 'required', isActive: true })
    .get();

  const employees = await db.collection('opendb-pf-employees').get();
  const period = '2026-03';

  for (const emp of employees.data) {
    for (const task of tasks.data) {
      const target = task.targetByRole?.[emp.role];
      if (target) {
        await db.collection('opendb-pf-employee-tasks').add({
          employeeId: emp._id,
          taskId: task.taskId,
          target,
          period,
          createdAt: Date.now(),
          updatedAt: Date.now()
        });
      }
    }
  }
}
```
